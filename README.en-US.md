# Smart-Press
- Fast and efficient Markdown website making tool
- Based on Vue3.0 SSR technology

## Getting Started
Manually
```bash
# install 
npm install smarty-press -g

# start to writing 
spress start

# build to html
spress build

```


## Development

```js
# clone first
npm link
cd demo
spress start
```

```bash
yarn add vue@3.0.0-beta.10

```
